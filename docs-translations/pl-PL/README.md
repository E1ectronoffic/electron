Proszę upewnij się, że korzystasz z dokumentacji, która odpowiada twojej wersji Electron. Numer wersji powinien być częścią URL strony. Jeżeli tak nie jest, być może używasz dokumentacji nie odpowiadającej twojej wersji Electron. Aby zobaczyć starszą wersję dokumentacji [przejrzyj tagi](https://github.com/electron/electron/tree/v1.4.0).
